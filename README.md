# A Django Learning Project

## About This Project
This project was built with the help of a Net Ninja course. It served as my learning companion, a place where I applied concepts that were taught in the course and built a functional Django application from scratch. 

** This is a learning project and not an original production application. The purpose was to understand Django funcdamentals through hands-on practice.

## What I Learned
- Django project structure and MVT pattern
- Models, views, and URL routing
- Template inheritance
- Class-based and function-based view

## 🛠️ Tech Stack

- **Backend:** Django, Python 
- **Frontend:** HTML, CSS, Bootstrap
- **Other:** Django Forms
